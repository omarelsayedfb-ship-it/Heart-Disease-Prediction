# Heart Disease Prediction 🩺❤️

مشروع تعلم آلي (Machine Learning) للتنبؤ بأمراض القلب باستخدام عدة خوارزميات تصنيف للمقارنة بين نتائجها.

## 📝 عن المشروع
تم تحليل بيانات المرضى واستخدام تقنيات معالجة البيانات (Data Preprocessing) مثل الـ Label Encoding للبيانات النصية، ثم تدريب الموديلات المختلفة.

## 🚀 المكتبات المستخدمة
* **Pandas & NumPy**: لمعالجة البيانات.
* **Matplotlib & Seaborn**: للرسوم البيانية وتحليل البيانات (EDA).
* **Scikit-Learn**: لبناء الموديلات وتدريبها.

## 📊 الخوارزميات المستخدمة والنتائج
بناءً على الكود الخاص بي، تم تجربة الخوارزميات التالية:
* **K-Nearest Neighbors (KNN)**: Accuracy ~68%
* **Decision Tree**: Accuracy ~74%
* **Gaussian NB**: Accuracy ~74%
* **Random Forest**: Accuracy ~78% 🏆
* **Gradient Boosting**: Accuracy ~78% 🏆

## 📁 محتويات المستودع
* `heart.csv`: ملف البيانات المستخدم.
* `tss.ipynb`: ملف الـ Jupyter Notebook الذي يحتوي على الكود والتحليلات.
